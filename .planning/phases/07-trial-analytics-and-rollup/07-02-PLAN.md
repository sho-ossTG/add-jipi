# Plan 07-02: Temporary Session View

## Goal

Add temporary session snapshots for operator visibility with TTL pruning.

## Tasks

- Implement session identity hashing from trusted IP and user-agent.
- Store session snapshots with TTL and maintain active session index.
- Wire writes into stream route outcomes (hit/success/degraded/error).

## Definition of Done

- Session snapshots are ephemeral and self-expire.
- Active session count can be derived from TTL-pruned index.
